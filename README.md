# DEVCONF 2026 Landing Page

A modern conference landing page created for **DEVCONF 2026**. The website presents conference information, featured speakers, ticket plans, venue details, and social media links.

## Features

* Modern conference landing page
* Conference hero section
* Featured speakers section
* Three pricing plans
* Venue information and directions
* Social media icons
* Smooth scrolling
* Button and card hover effects
* Semantic HTML structure

## Technologies Used

* HTML5
* CSS3
* Google Fonts
* Font Awesome
