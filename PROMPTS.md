Section structure:
- Add a centered heading
- Add a short subtitle
- Below the heading, create a two-column layout. 

Left column:
- Display a large, high-quality conference venue image.
- Use rounded corners.
- Add a small floating date badge over the image

Right column:
- Add the venue name 
- Add the address
- Include three small information rows with simple outline icons:
  1. Easy access by public transportation
  2. Nearby hotels and restaurants
  3. Accessible parking and entrance
- Add a short paragraph describing the venue.
- Include two buttons:
  Primary button: “Get Directions”
  Secondary outline button: “Explore Nearby Hotels”


Design requirements:
- Use container class name.
- Use dark navy headings, muted gray body text, and #2563EB for buttons and highlights.
